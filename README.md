# OrganDonor

Download mp3 from YouTube, upload to Google Play Music

## Usage
```shell
python organdonor.py https://www.youtube.com/watch?v=U4E60Ffa9yQ
```

## Name
YouTube to MP3 -> MP3 Pipe -> Pipe -> Organ -> Organ Donor is a cool song -> Sounds great
